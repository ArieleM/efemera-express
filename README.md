###  :play_or_pause_button: Instalação
Clone o repositório
```
git clone https://github.com/ArieleM/efemera-express.git
```
Abra a pasta PokeStore
```
cd efemera-express
```
Rodar o npm para instalar as dependências
```
npm install
```
Inicie o projeto 
```
npm start
```
